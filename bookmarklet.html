Add to Watchlist – Bookmarklet

When you're on an IMDb title page, click this bookmarklet to add the title to your watchlist.

Setup:

  1. Drag the link below to your bookmarks bar, or right‑click → "Add to bookmarks"
  2. Edit the bookmark and replace YOUR-SITE-URL with your watchlist URL (e.g. https://watchlist-trailers.netlify.app)
  3. If you already have the bookmark: delete it and drag again — old bookmarks redirect you away from IMDb; the new one opens the watchlist in a separate tab.
Add to Watchlist

How it works:

  • Runs only on IMDb title pages (e.g. imdb.com/title/tt1234567/)
  • Extracts title, year, and IMDb ID from the page
  • Opens your watchlist in a new tab (you stay on IMDb)
  • If the title is in your catalog and you're signed in, it's added to To Watch
  • If not in catalog, you'll see the add-movie command to run

Change the URL: Edit the bookmark and replace watchlist-trailers.netlify.app with your actual watchlist URL if different.
